# 视频工坊 (Video Workshop) v3.0

11 人双层 AI 视频创作专家团：Layer 1 GPT Image 2 SCULPT 关键帧资产库 → Layer 2 Seedance 2.0 视听一体全能参考分镜。
Seedance 2.0 原生支持 `()`音乐 `<>`音效 `{}`对白 `【】`字幕 一体化生成——音频是 prompt 的一部分，不是后期添加。

## 类型

Team 型（11 角色协作团队）

| 角色 | 花名 | 职责 |
|------|------|------|
| 主理人 | 画统筹 | 导演决策、类型检测、路由分发、全流程调度 |

## 创作架构

```
剧本 → 节拍图谱 + 脚本拆解 → Global Lookbook Code（DI调色总监） → 9类资产SCULPT规划
     → 动作编排蓝图（动作导演） → GPT Image 2关键帧 → 连续性审查
     → 声音+配音嵌入时序表 → Seedance 2.0视听一体分镜 → 合规审查 → 打包
```

包含：导演看板 / 角色 / 场景 / 动作看板 / 道具 / 光影 / 色彩脚本 / 关键帧 / 文字排版 9类资产体系。

## v3.0 更新

- **新增 DI调色总监（墨色之）**：Global Lookbook Code 全链路注入，三层光影矩阵（动机光场/大气介质/胶片印片），消除"每镜头不同光影"问题
- **新增 动作导演（武序之）**：关键姿态链（KP0→KP4）三维量化设计 + AI安全三级扫描（🟢🟡🔴）+ 库里肖夫蒙太奇拆招（局部替代/因果反应/环境隐喻）
- **9人 → 11人全线升级**：调色和动作从 prompt-engineer 的附属功能独立为专业角色
- **系统8（动作）+ 系统9（调色）** 完整内嵌至 agent 定义

## 使用示例

- 帮我从一段故事创意，先生成GPT Image 2关键帧资产库，再输出Seedance 2.0全能参考分镜包
- 我有一段故事，按完整叙事流程做成短片：剧本→拆解→关键帧→Seedance分镜
- 分析我的PRD文档，自动识别类型并运行完整的双层创作管线

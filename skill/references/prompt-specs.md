## 1. 情绪具象外化完整对照表

严禁使用抽象情绪词，必须外化为具体身体细节与物理动作。

**官方对照表（必须遵守，以下5项为官方原文）**：

| 抽象情绪 | 替换为具体外化表述（官方原文） |
|----------|----------------------------|
| 悲伤 | 低头、肩膀微微颤抖、眼眶泛红、手指无意识地攥紧衣角、泪水在眼眶里打转但没有落下 |
| 喜悦 | 嘴角抑制不住地上扬、眉眼舒展、脚步变得轻快、下意识地哼起小曲、忍不住原地转个圈 |
| 紧张/焦虑 | 频繁地看手表、手指不停敲击桌面、呼吸急促、眼神闪躲、无意识地啃咬指甲 |
| 愤怒 | 双拳紧握、下颌线紧绷、胸口剧烈起伏、眼神如刀般锐利、从牙缝里挤出话语 |
| 释然 | 长长地舒了一口气、紧绷的肩膀完全放松下来、脸上露出久违的、淡淡的微笑、抬头望向远方 |

|----------|---------------------------|

## 2. T.S.A.E.C.L.M.D 完整框架参考

### [C] 镜头与运镜清单

| 类别 | 类型 | 说明 |
|------|------|------|
| 景别 | 广角(24mm)、人像(85mm)、变形宽银幕、微距 | 选择一种 |
| 水平运动 | 左摇/右摇(Pan)、左移/右移(Truck) | - |
| 垂直运动 | 上摇/下摇(Tilt)、升降/下降(Pedestal) | - |
| 深度运动 | 推近/拉远(Zoom)、推进/拉出(Dolly)、滑动变焦(Dolly Zoom/眩晕效果) | - |
| 旋转运动 | 荷兰角(Dutch Angle)、360环绕(Orbit) | - |
| 空中运动 | 无人机上升、俯拍鸟瞰、低角度虫眼 | - |
| 跟拍 | 跟随镜头(Tracking)、引导镜头(Leading) | - |

**每个镜头只给一个主导运镜**——同时要求多种运镜最容易导致失稳。

### 运镜 → 情绪映射表

| 运镜 | 情绪/效果 | 使用场景 |
|------|-----------|----------|
| 推近(Dolly In) | 放大情绪、紧张加剧、揭示 | 角色内心变化、关键发现 |
| 拉远(Dolly Out) | 孤独感、疏离、揭示环境 | 离别、孤立、宏大揭示 |
| 环绕(Orbit) | 主角感、戏剧张力 | 对峙、高光时刻 |
| 跟拍(Tracking) | 代入感、紧张追逐 | 行走对话、追逐戏 |
| 上摇(Tilt Up) | 气势、威严、仰望 | 角色登场、建筑展示 |
| 下摇(Tilt Down) | 压迫、无力感 | 俯视、压迫场景 |
| 希区柯克变焦(Dolly Zoom) | 心理冲击、眩晕 | 惊吓、认知颠覆 |
| 手持晃动 | 紧张、真实感、混乱 | 战斗、灾难、纪录片风格 |
| 静态定镜 | 冷静、客观、凝视 | 对话、沉思、产品展示 |
| 荷兰角(Dutch Angle) | 不安、失衡、诡异 | 心理扭曲、梦境 |

### 景别 → 叙事功能

| 景别 | 叙事功能 |
|------|----------|
| 远景/大远景 | 交代环境、建立世界观 |
| 全景 | 展示人物全身与空间关系 |
| 中景 | 交代动作、推进叙事 |
| 近景 | 传递情绪、聚焦反应 |
| 特写/大特写 | 放大细节、强化冲击 |

### [L] 官方光照类型

黄金时刻(Golden Hour)、蓝色时刻(Blue Hour)、阴天漫射光、影棚灯光、轮廓光(Rim Light)、明暗对比(Chiaroscuro)、霓虹灯、烛光、伦勃朗光、丁达尔效应、霓虹漫反射。

### [A] 肢体三维度量化

每个动作必须包含：
- **幅度**（大/中/小）：动作的空间范围
- **速度**（快/中/慢）：动作的时间节奏
- **力度**（强/中/弱）：动作的发力程度

示例："大幅度挥舞双臂" vs "小幅度点头"；"缓慢抬起右手" vs "快速转头"；"重重地跺脚" vs "轻轻地放下杯子"

优先低缓连续小动作——Seedance 2.0 对缓慢连续的小动作表现更稳定；剧烈快速动作容易引发画面撕裂。
动作过渡衔接必须写明——如"从站立缓缓蹲下"而非"站着，然后蹲下"。

## 3. Negative Prompt 场景化补充完整清单

### 基础负面词（每个分镜必须包含）
```
low quality, blurry, deformed, mutated hands, extra limbs, bad anatomy, watermark, text, logo, child, loli, gore, violence, blood, mosaic, black bars, ugly face, poorly drawn, jittery motion, inconsistent lighting, physics error, frame skip, artifacts
```

### 场景化补充（按需追加）

**产品/电商：**
```
no logo distortion, no text artifacts, no packaging collapse, no duplicate product, no label blur
```

**人物/UGC：**
```
no extra fingers, no face drift, no lip mismatch, no background warping, no hand deformation
```

**电影感镜头：**
```
no shaky camera, no object melting, no random text, no muddy lighting, no flat blacks
```

## 4. 12类常见问题避坑速查表

| 问题 | 典型现象 | 根因 | 解决方案 |
|------|----------|------|----------|
| **人物ID漂移** | 同一角色在不同镜头中面部特征不一致，或中途"换脸" | 参考图混用、人脸占比过小、使用多视图 | ①准备人脸特写大头照（仅含面部，无表情最佳）；②提示词中分开定义——面部特征参考图片1（大头照），妆造参考图片2（全身照）；③重要素材越靠前权重越高；④**严禁使用人物多视图/三视图** |
| **视频含字幕** | 生成的视频画面中自动出现无关文字 | 提示词未显式排除 | 在提示词正文末尾加入官方约束词；若参考素材含文字先去除再使用；优先横屏生成（横屏字幕概率低于竖屏） |
| **Logo/水印** | 画面中生成品牌标志或半透明水印 | 模型训练数据包含带水印素材 | 在提示词末尾加入：`不要生成Logo。` `不要生成水印。` |
| **风格漂移** | 生成过程中画面风格逐渐偏离设定（如仙侠→真人） | 参考图写实 + 提示词风格描述不够 | ①加入明确风格约束词（如"3D国漫CG仙侠风格"）；②先将参考图变为目标风格再进行生视频 |
| **延长视频衔接跳变** | 延长后的视频与原视频衔接处有明显跳帧 | 编码边界帧不连续 | ①前一段视频末尾删减6帧；②后一段视频开头删减1帧；③续写时以转场切镜时刻结尾，下一段以新场景为起始 |
| **双胞胎问题** | 画面中出现两个外观相同的分身人物 | 多视图参考 + 主体定义不清 | ①清晰定义每个角色+标注对应参考图（如`张三（对应图片1）`）；②末尾加入全局约束：`视频全程禁止出现外形、着装、配饰完全一致的人物，禁止生成同款分身、双胞胎效果，同一画面中仅保留单个对应人物，不出现人物重复复刻。`；③优先单人独立照片，**严禁三视图/多视图**；④精简提示词，不直接用完整剧本 |
| **延长视频画质劣化** | 多次延长后画面质量逐段下降，人脸出现斑驳色块 | 多代编码累积损失 | ①将原始视频转为白模视频再续写；②优先用高清图片做参考；③控制续写次数 |
| **特效不符预期** | 指定特效（如爆炸、倒计时动画）未生成或效果偏差 | 纯文字描述不确定性 | ①用参考视频定义特效；②提示词中特效描述放在动作之前（提高权重） |
| **参考人物过多** | 当参考人物数量超过4人时，模型输出稳定性下降，可能出现人物数量不符或人物重复 | 模型多主体跟踪能力有限 | ①分步生成图片：将人物分组，确保每组生成的图片中人物数量不超过4人；②图片生成视频：使用第一步生成的多张分组图片作为参考素材，再生成最终视频 |
| **视频结尾噪音** | 视频最后几帧出现突兀咔哒声、截断杂音 | 模型生成边界不稳定 | ①**重新生成视频**；②使用剪映音量包络线做音频淡出处理 |
| **中文发音不准** | 人物口型对应的中文发音模糊或错误 | 多音字/生僻字/形近字识别偏差 | ①替换为发音一致的常用同音字（如"螭龙山"→"吃龙山"）；②注意该方案仅为优化手段，无法完全规避所有发音问题 |
| **音色参考不准** | 上传的音频参考未能准确复现音色 | 音频特征提取不稳定 | ①在提示词中补充细致的音色特征描述（如"低厚温润带细碎颗粒感中年男声"）；②保持视频台词风格与参考音频的语气、表述风格相近 |

## 5. 官方附录案例最佳实践快查表

| 场景类型 | 关键技巧 | 推荐模板 |
|----------|----------|----------|
| **文字生成（Slogan）** | 文字自动匹配画面风格和构图；生僻字建议先出图再导入 | `[场景描述] + 画面中出现文字"[内容]" + [风格/质感]` |
| **字幕** | 上传干净背景的人物参考图；字幕横向排列在画面下方 | `[镜头描述] + 画面底部出现字幕，字幕内容为"[台词]"，字幕需与音频节奏完全同步` |
| **气泡台词** | 适用于漫画/动画风格；气泡位置需与说话人物对齐 | `「角色」说：{台词}，角色说话时周围出现气泡，气泡里写着台词` |
| **图片参考-主体** | 按顺序上传；优先人脸特写大头照 | `将@图片1中的面部特征定义为主体1，@图片2中的全身造型作为主体1的服装参考` |
| **图片参考-3C数码** | 产品图片需白底/纯色背景，产品占画面足够面积 | `@图片1中的产品缓缓旋转展示，摄影棚柔光环境，镜头缓慢推近至产品细节` |
| **图片参考-家居** | 上传房间实拍图作为场景参考；保留空间布局但重置光影 | `以@图片1为场景参考，重新渲染光影质感，镜头从左至右缓慢横移展示空间全貌` |
| **Logo参考** | 上传PNG透明底Logo；提示词中指定Logo出现位置和动画方式 | `@图片1中的Logo从画面中心淡入浮现，微光粒子环绕，深色渐变背景` |
| **多主体参考** | 使用 @图片1 @图片2 分别锁定不同角色；每个主体独立定义 + 防分身约束 | `将@图片1中的角色定义为主体1（警察），将@图片2中的角色定义为主体2（小偷）。小偷在人群中惊慌狂奔，警察紧随其后全速追赶。` |
| **多宫格分镜图** | 上传含多个分镜的参考图；用"多宫格分镜图"关键词引导 | `参考@图片1的多宫格分镜图风格，按从左到右、从上到下的顺序依次生成对应画面` |
| **音频参考-音色** | 上传音频参考；提示词中补充音色特征描述 | `参考@音频1中的音色，生成...`；同时补充"低厚温润的中年男声"等特征描述 |
| **视频参考-动作** | 上传动作参考视频；提示词中单独描述动作，不混入角色描述 | `参考@视频1的动作风格，应用于@图片1定义的主体，在[场景]中执行[具体动作]` |
| **视频参考-运镜** | 上传运镜参考视频；在提示词中明确运镜类型和方向 | `参考@视频1的运镜方式，镜头[推近/拉远/环绕/…]，拍摄@图片1定义的主体` |
| **视频参考-特效** | 上传特效参考视频；提示词中将特效词汇放在动作之前提高权重 | `参考@视频1的粒子消散特效风格，主体从画面中心逐渐化为光点消散，慢动作` |
| **视频编辑-增元素** | 使用官方三段式：元素特征 + 出现时机 + 出现位置；直接写视频1引用 | `视频1的画面左侧，在花瓣飘落的同时，增加一棵发光的樱花树，画面其余部分保持不变` |
| **视频编辑-删元素** | 直接写视频1引用；明确要移除的元素；**对保持不变的元素在提示词中加以强调** | `视频1移除画面右侧的红色车辆，背景自动填补为与周围一致的街道场景，其余建筑和人物保持不变` |
| **视频编辑-改元素** | 直接写视频1引用；使用官方句式 `严格编辑视频N，将其中的原特征修改为新特征` | `严格编辑视频1，将其中的蓝色杯子修改为金色高脚杯，杯身反射暖色灯光，人物动作保持不变` |
| **视频延长-向后** | 使用延长功能而非重新生成；提示词描述延续动作 | `向后延长视频1，人物继续向前行走，经过转角后消失在小巷深处` |
| **视频延长-向前** | 在现有视频之前增加前导内容；用于补充起因或入场 | `向前延长视频1，人物从远处缓缓走近，走到视频1的起始位置` |
| **轨道补齐** | 用于多段视频之间的过渡衔接；需提示词描述过渡画面 | `视频1，树叶落地的瞬间，激起金色粒子特效，一阵风吹过，接视频2` |

## 6. 官方完整实战案例

### 案例1：宿舍情感短剧（偏文戏/对话）

**素材准备：**
- @图片1：女主半身照
- @图片2：宿舍场景参考图
- @视频1：室内对话运镜参考（中景推拉或轻微摇移）
- @音频1：室内环境声或轻音乐

**提示词全文：**
```
@图片1中的女孩作为主角，@图片2作为宿舍场景风格参考，参考@视频1的运镜方式。

镜头1：傍晚时分，女孩@图片1脚步轻快地走到宿舍门口@图片2，镜头中景平稳跟拍，暖黄色日光从窗外洒进走廊，她在门口停顿一下，深呼吸，表情略带紧张。

镜头2：女孩@图片1推开门走进宿舍，镜头切到室内中景，舍友们一边整理书本一边抬头看向她，其中一人笑着问{考得怎么样呀，过了吗}，镜头在几人之间缓慢切换半身特写。

镜头3：女孩@图片1先低头露出落寞表情，镜头给到她的近景，随后她抬头憋不住笑意，哈哈大笑说{骗你们的}，舍友们追着打闹起来，镜头缓慢拉远，定格在宿舍内一片欢声笑语的全景画面。

保持无字幕。不要生成Logo。不要生成水印。
全程画面高清电影纪实风，色调温暖，光影柔和；人物面部稳定不变形，动作自然流畅，无卡顿无闪烁；环境音效与@音频1自然融合。
```

### 案例2：古风悬崖对手戏（偏动作/氛围）

**素材准备（官方）：**
- @图片1：红衣女主
- @图片2：黑衣刺客（对手）
- @图片3：悬崖竹林场景图
- @视频1：武打对决运镜参考
- @音频1：紧凑的鼓点或打斗音效

**提示词全文（官方原文）：**
```
@图片1的红衣女子作为女主，@图片2的黑衣女子作为对手，场景参考@图片3的悬崖竹林环境，整体运镜和动作节奏参考@视频1，背景音效与@音频1同步。

镜头1：傍晚，镜头从红衣女子@图片1侧面中景缓慢推进，她站在悬崖边拿起酒壶喝酒，衣袂在山风中轻轻摆动，镜头环绕她半圈，从正面推到背影，远处隐约可见竹林中的黑衣人影。

镜头2：镜头变焦渐隐到远景，无人机视角俯瞰整片悬崖和竹林，两人分立山崖两端，山风卷起衣摆和尘土，节奏随鼓点略微加快。

镜头3：镜头切回地面近景，二人缓慢拔剑对峙，红衣女子@图片1神情从漫不经心转为冷冽，黑衣女子@图片2目光坚毅，剑尖微微颤动，镜头平稳跟随两人绕圈移动，最后定格在两剑相交前一瞬间的特写。

整体画面烟雨江湖电影感，冷调低饱和，电影胶片质感，光影层次丰富；人物面部和身体比例稳定不变形，动作连贯自然，不僵硬，无穿模无卡顿。
```

**校验要点（写完后对照）：**
1. @引用是否与素材准备一致？
2. 每个镜头是否按四要素组织（运镜→动作表情→空间变化→音频）？
3. 镜头编号是否取代了绝对秒数？
4. 约束词/风格描述是否在末尾加入？
5. 多主体场景是否分别定义并使用标签区分？

## 7. 公式速查

### 基础公式（快速生成/探索阶段）
```
[主体] + [动作] + [镜头] + [场景] + [风格]
```
权重从高到低，主体权重最高。适合首次生成、方向探索、风格测试——投入成本低、迭代速度快。当基础公式产出已接近目标效果，需要精细控制时，再切换至进阶公式。

### 进阶公式（精细控制阶段）
```
精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质 + 约束条件
∣_____________∣  ∣_____________________________∣  ∣_______________________________∣
    必需（2项）              建议（3项）                      建议（3项）
```
先锁定"谁"在"干什么"（必需项），再交代"在哪""什么氛围""怎么拍"（建议项），最后用风格、画质和约束词收紧结果。

### 通用填充模板
```
[主体定义] 将图片N中的[特征]定义为主体名
[镜头1] [运镜方式] + [主体动作/表情] + [空间变化] + [音频信息]
[镜头2] [承接动作流]
[光影色调] [具体光线、色彩描述]
[视觉风格] [风格参照]
[画质要求] [分辨率、帧率等]
[约束条件] [禁止重复人物/禁止特定元素]
```

### 迭代三规则
1. 一次只改一个大变量（prompt文案 / reference选择 / 运镜指令 / 时长 / 画幅）
2. 先缩窄再加复杂度（更少动作 → 更简单镜头 → 更紧构图）
3. 让prompt决定模式（频繁出现"same face/same shape"时应切换为图生视频或参考驱动模式）

### 好Prompt四问（写完后自查）
1. 观众第一眼应该看到什么？
2. 镜头在做什么？
3. 画面属于什么视觉气质？
4. 哪些错误绝不能出现？

# Seedance 2.0 动作提示词速查表

> 来源：CSDN VarIsle Seedance 2.0 动作捕捉提效指南 + 搜狐 AI 视频四步法 + StudioBinder 影视调度理论  
> 格式：每类 = 模板ID + 适用场景 + 关键姿态链(KP0-KP4) + 三维量化 + 风格参数 + 常见陷阱  
> 版本 v1.0 / 2026-06-24

---

## 使用说明

每类动作模板提供：
- **KP链**：3-5个关键姿态，直接复制到 Seedance [A] 维度
- **三维量化**：幅度(大/中/小) + 速度(快/中/慢) + 力度(强/中/弱)
- **风格参数**：直接附加到提示词末尾的风格标签
- **避坑**：该动作类型最常见的生成失败模式及替代方案

## 第一类：基础移动——走/跑/跳

### A01 — 自然行走（Walk Natural）

| 项目 | 内容 |
|------|------|
| **KP0** | 双脚并立，重心居中，双臂自然下垂 |
| **KP1** | 右腿前迈，右膝屈15°，重心前移30%，左臂前摆20° |
| **KP2** | 右腿支撑，左腿后蹬，重心完全在右腿，双臂交替摆动 |
| **KP3** | 左腿前迈，左膝屈15°，重心前移30%，右臂前摆20° |
| **KP4** | 左腿支撑，右腿后蹬，双臂交替——回到KP2镜像 |
| **幅度** | 中（步幅约60-80cm） |
| **速度** | 中（约1.2步/秒） |
| **力度** | 弱（日常力度，无冲击感） |
| **风格** | `natural_casual_1.0x` |
| **避坑** | 避免"机械行军"——确保双臂与双腿对角交替；步幅过大(>1m)易导致腿部拉伸变形 |

### A02 — 自信行走（Walk Confident / Power Walk）

| 项目 | 内容 |
|------|------|
| **KP0** | 直立，挺胸，双肩后收，下巴微抬5° |
| **KP1** | 右腿大幅度前迈，右膝屈10°，重心前移40%，双臂摆幅加大 |
| **KP2** | 右腿支撑，重心稳定，胸部保持前挺，视线水平 |
| **KP3** | 左腿大幅度前迈，镜像KP1 |
| **KP4** | 落步有力，鞋底接触地面带有微震感 |
| **幅度** | 大（步幅90-110cm） |
| **速度** | 中偏快（约1.5步/秒） |
| **力度** | 中（每步落地有轻微冲击） |
| **风格** | `natural_casual_1.3x` |
| **避坑** | 挺胸过度→上半身后仰→重心不稳；加一个"双肩后收"的KP0微动确保姿态锁定 |

### A03 — 奔跑（Run Sprint）

| 项目 | 内容 |
|------|------|
| **KP0** | 起跑预备：左脚前右脚后，身体前倾15°，双臂屈肘90° |
| **KP1** | 右脚蹬地，身体前倾25°，左膝高抬至髋部高度，右臂前摆 |
| **KP2** | 空中腾跃：双脚离地，身体完全伸展，前倾30° |
| **KP3** | 左脚落地，膝盖微屈缓冲，右脚离地后摆，左臂前摆 |
| **KP4** | 回到KP1镜像：右脚前迈，循环 |
| **幅度** | 大（步幅1.5-2m，腾空高度10-20cm） |
| **速度** | 快（约3步/秒） |
| **力度** | 强（每次落地冲击力明显） |
| **风格** | `athletic_explosive_1.5x` |
| **避坑** | 跑步最容易出现"脚底抹油"（滑步无腾空）——必须在KP2明确标注"双脚离地"；速度过快(>4步/秒)易撕裂 |

### A04 — 跳跃（Jump Vertical）

| 项目 | 内容 |
|------|------|
| **KP0** | 预备下蹲：膝盖屈60°，双臂后摆至身后，脊柱前倾10° |
| **KP1** | 爆发蹬地：膝盖完全伸展，脚尖离地，双臂猛烈上摆至头顶 |
| **KP2** | 腾空顶点：身体完全伸展，脚尖下垂，双手举过头顶，头部微仰 |
| **KP3** | 下落：双臂开始下放，膝盖微屈预备缓冲，脚尖朝下 |
| **KP4** | 落地缓冲：膝盖屈至45°，重心下沉，双臂前摆恢复平衡 |
| **幅度** | 大（腾空30-80cm） |
| **速度** | 快（KP0→KP1爆发阶段） |
| **力度** | 强（蹬地瞬间 + 落地冲击） |
| **风格** | `athletic_explosive_1.8x` |
| **避坑** | 落地无缓冲会显得僵硬（"砖块落地"）——KP4的膝盖屈+重心下沉不可省略；腾空高度>1m容易导致身体比例失调 |

### A05 — 坠落（Fall / Drop）

| 项目 | 内容 |
|------|------|
| **KP0** | 失衡瞬间：身体后倾超15°，双臂挥舞试图抓握，表情惊恐 |
| **KP1** | 开始坠落：身体完全失衡，与地面成45°，双臂上伸 |
| **KP2** | 自由落体：身体水平或微卷，衣物向上飘动（空气阻力），头发上浮 |
| **KP3** | 着地预备：身体蜷缩，双臂护头，膝盖屈至胸前 |
| **KP4** | 冲击瞬间：身体触地，灰尘扬起，衣物/头发惯性下压，面部肌肉抖动 |
| **幅度** | 大 |
| **速度** | 由慢到快（KP0-KP1慢，KP2-KP4加速） |
| **力度** | 强 |
| **风格** | `cinematic_slowmo_0.4x`（慢动作坠落）/ `athletic_explosive_1.2x`（实速坠落） |
| **避坑** | KP2阶段衣物必须向上飘动（空气阻力）否则像在真空中坠落；KP4的灰尘/冲击反馈是关键真实感来源 |

## 第二类：上肢动作——手/臂

### A06 — 挥手（Wave Hand）

| 项目 | 内容 |
|------|------|
| **KP0** | 手臂置于体侧或胸前，手掌张开，手指自然并拢 |
| **KP1** | 手肘屈90°，前臂抬起至与地面平行，手掌朝前 |
| **KP2** | 手掌左右摆动：腕关节尺偏→桡偏，幅度20°，频率2-3次/秒 |
| **KP3** | 手臂抬高至头顶上方，手掌加大摆动幅度至30° |
| **KP4** | 手臂缓缓放下，回到体侧或继续挥动 |
| **幅度** | 中（腕关节偏转15-30°） |
| **速度** | 中（2-3次/秒） |
| **力度** | 弱 |
| **风格** | `natural_casual_0.9x` |
| **避坑** | 手臂僵硬"像机器人"——必须在KP1和KP2之间加入前臂的旋转过渡，确保肘关节不是锁死的 |

### A07 — 指向（Point Finger）

| 项目 | 内容 |
|------|------|
| **KP0** | 手臂自然下垂或置于体侧 |
| **KP1** | 肩关节屈曲，手臂从体侧抬起，肘关节伸展中 |
| **KP2** | 手臂完全伸展，食指伸出，其余四指蜷握，指尖指向目标方向 |
| **KP3** | 锁定：手臂保持伸展，指尖稳定在目标点，视线跟随指尖方向 |
| **KP4** | 回收：手臂缓缓放下，食指收回 |
| **幅度** | 大（肩关节屈曲90-180°） |
| **速度** | 中偏快（抬手指向阶段0.5-0.8s） |
| **力度** | 中（指向瞬间有停顿感） |
| **风格** | `martial_precision_medium`（精准指向）/ `natural_casual_1.0x`（随意指向） |
| **避坑** | 手指常被渲染成"多指/少指"——提示词中明确指定"食指伸出，其余四指蜷握"，不要笼统写"用手指指向" |

### A08 — 抓取（Grasp Object）

| 项目 | 内容 |
|------|------|
| **KP0** | 手靠近目标物体上方或侧方10-15cm，手掌张开，五指微屈 |
| **KP1** | 手向物体移动，五指完全张开（预抓取姿态） |
| **KP2** | 手指合拢，指尖→指腹→手掌依次接触物体表面 |
| **KP3** | 握紧：手指完全包裹物体，指关节屈曲至接触物体轮廓，手腕微调角度 |
| **KP4** | 提起/移动：手腕保持稳定，手臂开始移动物体 |
| **幅度** | 小（手指屈曲约30-60°） |
| **速度** | 慢（抓取阶段0.8-1.2s） |
| **力度** | 中（手部有握力感） |
| **风格** | `natural_casual_1.0x` |
| **避坑** | 手与物体穿插/穿透——KP1中必须保持手掌与物体间可见间隙；手指包裹物体时易出现"多指融合" |

### A09 — 投掷（Throw Object）

| 项目 | 内容 |
|------|------|
| **KP0** | 持物站立，手臂屈肘90°，手紧握物体置于肩侧或头侧 |
| **KP1** | 身体向后扭转（脊柱旋转20-30°），持物手臂后拉，对侧手臂前伸平衡 |
| **KP2** | 爆发投出：身体向前旋转，肩关节快速屈曲+内旋，肘关节伸展，手腕在释放点瞬间弹射 |
| **KP3** | 释放瞬间：手掌张开，物体离手，手指指向目标方向 |
| **KP4** | 惯性跟随：手臂继续前摆，身体重心前移，后脚踮起 |
| **幅度** | 大 |
| **速度** | 极快（KP2爆发阶段<0.3s） |
| **力度** | 强 |
| **风格** | `athletic_explosive_1.8x` |
| **避坑** | 物体离手后飞行轨迹需要在prompt中单独描述（"物体沿抛物线飞出，旋转中"），否则物体可能粘在手上 |

### A10 — 拥抱（Embrace / Hug）

| 项目 | 内容 |
|------|------|
| **KP0** | 面向对方，距离约50cm，双臂微张，表情温暖 |
| **KP1** | 双臂从两侧展开至与肩同宽，手掌朝前，身体微前倾 |
| **KP2** | 双臂环绕对方背部/肩部，手掌贴附对方身体，头微侧靠向对方 |
| **KP3** | 拥抱收紧：双臂施加轻微压力（衣物出现轻微褶皱），眼睛闭合或微眯 |
| **KP4** | 缓缓松开，双臂回收，身体后退半步，眼神对视 |
| **幅度** | 大（双臂展开120-180°→合拢） |
| **速度** | 慢（整个拥抱2-4s） |
| **力度** | 中（拥抱有收紧感但不勒紧） |
| **风格** | `natural_casual_0.8x` / `cinematic_slowmo_0.6x`（情绪化拥抱） |
| **避坑** | 双人拥抱最大陷阱：手部混淆（谁的胳膊在上/在下）——必须明确标注"角色A右臂在角色B左肩上方"；脸颊贴附时避免面孔融合 |

## 第三类：下肢动作——腿/脚

### A11 — 踢腿（Kick Forward）

| 项目 | 内容 |
|------|------|
| **KP0** | 格斗站姿：双脚肩宽，膝盖微屈，双手防护面部，重心居中 |
| **KP1** | 后脚蹬地，重心移至前脚，踢击腿屈膝上提至髋部高度 |
| **KP2** | 踢击腿膝关节快速伸展，脚背绷直，脚尖指向目标，髋关节屈曲至90°+ |
| **KP3** | 冲击瞬间：脚背/胫骨接触目标，力量传递，目标表面产生冲击波纹 |
| **KP4** | 快速回收：踢击腿屈膝收回，落地恢复格斗站姿 |
| **幅度** | 大（踢击腿髋屈90-180°） |
| **速度** | 极快（KP1→KP2爆发<0.2s） |
| **力度** | 强 |
| **风格** | `martial_precision_high` |
| **避坑** | 踢腿最怕"面条腿"（没有伸展爆发感）——KP1到KP2的膝关节伸展速度必须极快；KP3冲击反馈是关键 |

### A12 — 深蹲（Squat Deep）

| 项目 | 内容 |
|------|------|
| **KP0** | 直立站立，双脚与肩同宽，脚尖微外八，双臂前伸保持平衡 |
| **KP1** | 髋关节后移（像坐椅子），膝盖屈至45°，脊柱保持中立，重心下沉 |
| **KP2** | 继续下蹲：膝盖屈至90°-120°，髋部低于膝盖，脚跟着地，双臂前伸 |
| **KP3** | 底部停顿：大腿后侧贴小腿，重心完全在脚跟，核心收紧 |
| **KP4** | 蹬起：臀大肌发力，膝盖伸展，髋关节前推，回到直立 |
| **幅度** | 大（髋关节-30°至120°） |
| **速度** | 慢（下蹲2s + 停顿0.5s + 起立2s） |
| **力度** | 强（下肢肌群发力明显） |
| **风格** | `athletic_explosive_1.0x` |
| **避坑** | 膝关节超伸——KP2中明确"膝屈90-120°"；脚后跟必须着地否则像穿高跟鞋深蹲 |

### A13 — 转身（Turn Around）

| 项目 | 内容 |
|------|------|
| **KP0** | 正面朝前站立 |
| **KP1** | 头部先转：颈椎旋转30°，视线先于身体转向目标方向 |
| **KP2** | 身体跟随：脊柱旋转，右脚（以右转为例）外旋45°，左脚为轴 |
| **KP3** | 旋转过半：身体侧对镜头，左脚踮起跟随旋转，重心在两脚间 |
| **KP4** | 完成转身：身体180°背对镜头，双脚并拢，姿态稳定 |
| **幅度** | 大（全身180°旋转） |
| **速度** | 中（慢转身1.5-2s / 快转身0.5-0.8s） |
| **力度** | 弱-中（日常转身弱，警觉转身中） |
| **风格** | `natural_casual_1.0x`（日常）/ `martial_precision_medium`（警觉） |
| **避坑** | 头身同步旋转像"机器人转盘"——必须KP1头先转、KP2身体再跟；旋转中衣物应有轻微惯性摆动 |

### A14 — 跪地（Kneel Down）

| 项目 | 内容 |
|------|------|
| **KP0** | 站立，表情决绝/虔诚/疲惫（根据情境） |
| **KP1** | 单膝下沉：右膝（以右膝为例）缓缓弯曲，右脚尖后移，重心下沉 |
| **KP2** | 右膝触地：右膝接触地面，左腿屈膝90°支撑，双手自然垂放或撑地 |
| **KP3** | 双膝着地：左膝也缓缓放下触地，身体重心完全下沉，上半身保持直立或微前倾 |
| **KP4** | 静止跪姿：双膝着地，脚背贴地，双手垂放大腿或合十，头部微低 |
| **幅度** | 大（身体下降50-80cm） |
| **速度** | 慢（1.5-3s） |
| **力度** | 弱-中（下降轻柔，触地有轻微冲击） |
| **风格** | `cinematic_slowmo_0.5x`（情绪化跪地） |
| **避坑** | KP2膝盖触地必须有冲击感和衣物褶皱变化；衣物下摆应随身体下降自然垂落 |

### A15 — 跳跃落地（Jump Landing）

| 项目 | 内容 |
|------|------|
| **KP0** | 空中姿态：身体微蜷，手臂微张保持平衡，双脚朝下 |
| **KP1** | 触地瞬间：脚尖→脚掌→脚跟依次触地，膝盖屈15-20°缓冲 |
| **KP2** | 深度缓冲：膝盖屈至45-60°，身体前倾10°，双臂前摆稳定重心 |
| **KP3** | 回弹恢复：膝盖缓缓伸展，身体恢复直立，重心稳定 |
| **KP4** | 落地完成：稳定站姿，呼吸微促，身体无晃动 |
| **幅度** | 大（膝盖缓冲幅度大） |
| **速度** | 快→慢（下降快，缓冲后慢停） |
| **力度** | 强（冲击吸收） |
| **风格** | `athletic_explosive_1.2x` |
| **避坑** | 落地无缓冲=跳楼——KP1-KP2的膝盖屈曲是核心，不能省略；添加地面微尘扬起增强真实感 |

## 第四类：全身复合动作

### A16 — 格斗出拳（Fight Punch）

| 项目 | 内容 |
|------|------|
| **KP0** | 格斗站姿：双脚与肩宽，左脚前右脚后，双拳护颚，肘部贴身 |
| **KP1** | 蓄力扭转：脊柱旋转20°，后手拳收回腰侧，前手保持防护，后脚蹬地 |
| **KP2** | 爆发出拳：髋关节旋转带动脊柱→肩关节→肘关节依次伸展，拳面旋转至水平，后脚踮起 |
| **KP3** | 打击瞬间：拳面接触目标，手臂完全伸展，身体重心前移至65%，面部肌肉微震 |
| **KP4** | 快速回收：出拳手臂闪电回收至护颚位置，身体回正恢复站姿 |
| **幅度** | 大（出拳臂伸展至最大长度） |
| **速度** | 极快（KP1→KP3<0.25s） |
| **力度** | 极强 |
| **风格** | `martial_precision_high` |
| **避坑** | 出拳最常见问题：(1)不出拳=手臂没伸展 (2)棉花拳=无冲击反馈 (3)多臂=角色混淆。必须在KP3明确冲击反馈+KP4回收速度 |

### A17 — 格斗闪避（Fight Dodge）

| 项目 | 内容 |
|------|------|
| **KP0** | 格斗站姿，视线锁定对手 |
| **KP1** | 察觉攻击：瞳孔微缩，头部微后撤，身体重心后移预备 |
| **KP2** | 闪避动作：上半身快速侧倾/后仰/下潜，幅度30-50cm，重心完全偏移 |
| **KP3** | 闪过瞬间：攻击从身体旁掠过（可描述气流/衣物飘动），面部有风压反应 |
| **KP4** | 恢复：身体回弹至格斗站姿，重心稳定，呼吸微促 |
| **幅度** | 大（身体偏移30-50cm） |
| **速度** | 极快（闪避动作<0.3s） |
| **力度** | 中（闪避轻盈，回弹迅速） |
| **风格** | `martial_precision_high` |
| **避坑** | 闪避和出拳容易混淆主体边界——如双人对打，必须用显式标签绑定区分角色A（出拳）和角色B（闪避） |

### A18 — 舞蹈旋转（Dance Pirouette）

| 项目 | 内容 |
|------|------|
| **KP0** | 预备站位：双脚四位(芭蕾)/并立(现代)，双臂呈弧形，核心收紧 |
| **KP1** | 起转：支撑腿立半脚尖，活动腿提至 passé 位（脚尖触膝），双臂收至一位 |
| **KP2** | 旋转中：身体绕纵轴旋转，头留至前方（Spotting），支撑腿半脚尖稳定 |
| **KP3** | 旋转360°：头部快速甩回前方（Spot），身体完成一整圈，双臂可逐渐打开 |
| **KP4** | 落地收束：活动腿放下至五位或并立，膝盖微屈缓冲，手臂呈弧形收势 |
| **幅度** | 大（360°+旋转，支撑腿立半脚尖15-20cm） |
| **速度** | 中（约1-1.5转/秒） |
| **力度** | 弱-中（轻盈、反重力感） |
| **风格** | `ballerina_gravity_0.5x` |
| **避坑** | 旋转中最容易"糊成一团"——必须在KP1明确"支撑腿立半脚尖"，这是旋转的核心力学支点；Spotting（留头甩头）是区分"转圈"和"芭蕾转"的关键 |

### A19 — 倒地（Collapse / Fall to Ground）

| 项目 | 内容 |
|------|------|
| **KP0** | 意识丧失/受伤：眼神涣散，身体失衡前兆，肌肉突然松弛 |
| **KP1** | 开始倾斜：身体向一侧倾倒，膝盖先行弯曲，头部下垂 |
| **KP2** | 倒地中途：身体与地面成30°-45°，双臂无力垂落，衣物开始因重力变形 |
| **KP3** | 触地：身体侧面或正面接触地面，产生轻微弹跳，灰尘微扬，面部肌肉因冲击抖动 |
| **KP4** | 着地静止：身体完全躺平，四肢自然散开，胸腔微动呼吸，无自主运动 |
| **幅度** | 大（身体从站立到平躺） |
| **速度** | 中偏快（1-2s） |
| **力度** | 弱（不是"被推倒"而是"失去意识瘫倒"） |
| **风格** | `natural_casual_0.7x`（自然瘫倒）/ `cinematic_slowmo_0.3x`（戏剧化） |
| **避坑** | 最容易"僵尸倒"（像木板一样直挺挺倒）——KP1的膝盖先行弯曲是关键，"瘫倒"和"被推倒"的区别在此 |

### A20 — 翻滚（Roll Forward/Backward）

| 项目 | 内容 |
|------|------|
| **KP0** | 低姿预备：蹲姿，双手撑地，头内收以下巴触胸，背部弓起 |
| **KP1** | 蹬地启动：双脚蹬地，重心前移，肩膀即将触地，身体蜷缩成球 |
| **KP2** | 滚动中：背部→腰部→臀部依次接触地面，身体保持蜷缩，旋转180° |
| **KP3** | 滚出：双脚着地，身体开始伸展，双手离开地面寻找平衡 |
| **KP4** | 起身完成：蹲姿或半蹲，重心稳定，双手可展开保持平衡 |
| **幅度** | 大（身体360°旋转前移） |
| **速度** | 快（整个翻滚1-1.5s） |
| **力度** | 中（滚动有惯性，蹬地有力） |
| **风格** | `athletic_explosive_1.3x` |
| **避坑** | 翻滚最易"卡帧"——KP1→KP2的滚动阶段需要足够的角速度描述；头内收（下巴触胸）几乎是所有翻滚失败的原因（没写这个→头撞地） |

## 第五类：精微动作——表情/手势

### A21 — 微笑展开（Smile Emerge）

| 项目 | 内容 |
|------|------|
| **KP0** | 中性表情：面部肌肉放松，嘴角水平，眼轮匝肌无收缩 |
| **KP1** | 嘴角微动：口角提肌轻微收缩，嘴角上提2-3mm，眼轮匝肌开始微收 |
| **KP2** | 笑容成形：嘴角上提8-10mm，颧大肌收缩使脸颊上推，眼角出现细纹（鱼尾纹） |
| **KP3** | 完全微笑：嘴角上扬至最大幅度，牙齿微露（如适用），眼轮匝肌完全收缩，眼睛微眯 |
| **KP4** | 笑容保持：微表情稳定，眼周细纹可见，整体表情温暖自然 |
| **幅度** | 小（嘴角位移<10mm） |
| **速度** | 慢（从KP0到KP3约1.5-2.5s） |
| **力度** | 弱 |
| **风格** | `natural_casual_0.8x` |
| **避坑** | "假笑"（只动嘴角，眼睛不动）——KP2必须同步描述眼轮匝肌的收缩和鱼尾纹，这是区分真笑假笑的唯一视觉锚点 |

### A22 — 愤怒积累（Anger Build-up）

| 项目 | 内容 |
|------|------|
| **KP0** | 平静：面部放松，呼吸均匀 |
| **KP1** | 微表情预警：下颌线微收紧，嘴唇抿紧1mm，眉间肌轻微收缩 |
| **KP2** | 愤怒上升：双拳开始握紧，下颌线明显紧绷，胸口起伏幅度增加，鼻孔微张 |
| **KP3** | 爆发前兆：咬肌隆起，眉间深锁成"V"形，呼吸急促，颈静脉微凸，双手微颤 |
| **KP4** | 爆发瞬间：（接A16格斗或A23爆发性动作） |
| **幅度** | 小→中（面部肌肉收缩渐进） |
| **速度** | 由慢到快（KP0-KP2慢，KP3加速） |
| **力度** | 由弱到强 |
| **风格** | `natural_casual_0.6x→1.2x` |
| **避坑** | 情绪跃进（平静→暴怒无过渡）——KP1-KP3的渐进过程是关键；下颌线+眉间+呼吸三项必须同步升级 |

### A23 — 恐惧颤抖（Fear Trembling）

| 项目 | 内容 |
|------|------|
| **KP0** | 轻微不安：瞳孔微扩，呼吸频率微增 |
| **KP1** | 恐惧上升：瞳孔明显放大，面色苍白，嘴唇微颤，手部开始发抖（幅度2-3mm，频率3-5Hz） |
| **KP2** | 颤抖加剧：手部抖动幅度增至5-8mm，肩膀开始微颤，呼吸急促且不规则 |
| **KP3** | 全身颤抖：四肢明显抖动，牙齿轻微打颤，喉结上下滚动，后背紧贴墙壁（如有） |
| **KP4** | 恐惧顶峰：瞳孔缩至极限或极度放大，全身僵直或蜷缩，面部肌肉扭曲 |
| **幅度** | 由小到大（手部2mm→8mm） |
| **速度** | 颤抖频率3-8Hz |
| **力度** | 弱（非自主颤抖，非故意发力） |
| **风格** | `panic_erratic_1.5x` |
| **避坑** | 颤抖不是快速来回晃——是微小的、不规则的肌肉抽搐；频率过高(>10Hz)→模糊/撕裂；对手指/嘴唇这些高频抖动部位单独描述 |

### A24 — 手势对话（Gesture During Speech）

| 项目 | 内容 |
|------|------|
| **KP0** | 说话前静态：双手自然垂放或置于桌面，表情平静 |
| **KP1** | 手势起手：优势手从静止位置抬起至胸前或桌面高度，手掌微张 |
| **KP2** | 手势进行中：手掌随说话节奏做出"强调""列举""比划大小"等动作 |
| **KP3** | 手势高点：手势幅度最大（手掌在胸前20-30cm），与话语节奏同步 |
| **KP4** | 手势回收：手缓缓回到静止位置或过渡到下一个手势 |
| **幅度** | 中（手掌移动20-40cm） |
| **速度** | 与说话节奏同步（约0.3-0.5s/手势节拍） |
| **力度** | 弱-中 |
| **风格** | `natural_casual_1.0x` |
| **避坑** | 手势与说话节奏脱节（手已经放下了嘴还在说）——必须将手势的KP与对白{}的时间戳对齐；意大利人的手势幅度>北欧人的手势幅度 |

### A25 — 晕眩/失衡（Dizzy / Loss of Balance）

| 项目 | 内容 |
|------|------|
| **KP0** | 晕眩开始：视线模糊（眼球微颤），头部开始轻微晃动，重心微不稳 |
| **KP1** | 失衡步态：双脚间距自动加宽保持平衡，双臂微张寻找支撑，步态蹒跚 |
| **KP2** | 严重失衡：身体侧倾10-15°，手臂急切寻找墙壁/扶手/他人，膝盖微屈 |
| **KP3** | 几乎跌倒：身体侧倾>20°，手最终触到支撑物，重心完全依靠支撑 |
| **KP4** | 依靠支撑恢复：身体靠墙或扶稳，头部仍微晃，呼吸急促，视线仍模糊 |
| **幅度** | 中（身体侧倾10-25°） |
| **速度** | 慢（失衡过程2-4s） |
| **力度** | 弱（肌肉无力感） |
| **风格** | `panic_erratic_1.0x` |
| **避坑** | 失衡≠醉酒——区别在于"是否有意识挣扎"；KP1的"脚间距加宽"是身体的自然平衡反应，不能省略 |

## 第六类：特殊/特效动作

### A26 — 慢动作（Slow Motion Cinematic）

| 项目 | 内容 |
|------|------|
| **KP0** | 动作起点：同常规动作的KP0 |
| **KP1-KP4** | 与常规动作相同的KP链 |
| **差异** | 所有KP间距×2-4倍（如正常0.5s间隔→慢动作2s间隔） |
| **附加元素** | 衣物飘动更明显、发丝缓滞、水滴/尘埃可见轨迹、面部表情微变化逐帧可见 |
| **幅度** | 同原动作 |
| **速度** | 慢（原速度×0.25-0.5） |
| **力度** | 同原动作（但冲击感被拉伸，更有"重量感"） |
| **风格** | `cinematic_slowmo_0.3x` 至 `0.6x` |
| **避坑** | 慢动作最怕"一顿一顿"——必须确保所有KP之间的变化是连续平滑的；在prompt中额外添加"衣物/头发/环境粒子在缓慢飘浮中"来强化时间拉伸感 |

### A27 — 悬浮/失重（Levitation / Zero-G）

| 项目 | 内容 |
|------|------|
| **KP0** | 地面站立，身体开始有上升感 |
| **KP1** | 脚尖离地：脚尖轻轻离开地面2-5cm，衣物下摆开始上浮，头发失去重力约束 |
| **KP2** | 悬浮上升：身体缓缓上升至离地30-50cm，四肢自然微张，衣物完全失重漂浮 |
| **KP3** | 悬浮顶点：身体悬停在空中，头发四散漂浮，衣物如水中展开，周围可能漂浮小物件 |
| **KP4** | 悬浮微漂：身体在空中轻微上下/左右漂移（1-2cm），手臂缓慢划动如水中 |
| **幅度** | 大（离地30-50cm） |
| **速度** | 极慢（上升0.5-1m/s） |
| **力度** | 极弱（完全失重） |
| **风格** | `ballerina_gravity_0.1x` |
| **避坑** | 失重≠游泳——身体应是无主动发力的漂浮，而非游泳般的划水动作；衣物和头发的失重表现是区分"站桩"和"真悬浮"的核心 |

## 附录：动作质量速查

### 负面动作清单（12类易崩动作及替代）

| 易崩动作 | 失败模式 | 替代方案 |
|---------|---------|---------|
| 掰断/折断 | 断面崩坏、形变异常 | 改为"已断"的结果态，跳过多段断裂过程 |
| 剥瓣（橘子等） | 理解一般，常剥成碎块 | 改为"已剥好"的结果态，或替换为"切瓣" |
| 反常规横切（如肉夹馍水平切） | 模型有强先验（刀=竖切） | 改用结果态镜头（"两半已分离"），分镜≤6张 |
| 手指精细动作（织毛衣、穿针） | 手指粘连/多指/少指 | 特写+大数据量镜头，或改用手部大动作替代 |
| 多人打斗混战（≥3人） | 角色混淆、ID漂移、肢体交叉崩溃 | 分组拍摄（≤2人/组），分步生成 |
| 高速连击（>3次/秒） | 运动模糊过度、画面撕裂 | 降速到1-2次/秒，或用关键帧+慢动作替代 |
| 地面翻滚 | 中间帧卡顿、角度跳变 | 降低角速度描述，增加中间KP数量 |
| 挥剑/武器轨迹 | 轨迹不一致、弧度断裂 | 添加"挥动轨迹线"描述（如"剑尖在空中画出一道银色弧线"） |
| 水流/布料解算 | 物理错误、穿模 | 降低复杂度，用"飘动"替代"缠绕" |
| 面部极速变化（<0.5s从哭到笑） | 中间帧扭曲 | 增加过渡KP，间隔≥0.3s |
| 水中游泳 | 身体/水交界处模糊、比例失调 | 只拍上半身或水面以上的动作 |
| 镜面/反射中的动作 | 镜像可能不一致 | 避免镜面反射中的复杂动作，改用间接角度 |

### 动作-运镜互锁原则

| 主体动作方向 | 推荐运镜 | 效果 |
|------------|---------|------|
| 主体向右移动 | 跟摇向右（Pan Right） | 跟随主体，保持构图稳定 |
| 主体向前走 | 后退跟拍（Dolly Back）或推近（Dolly In） | 后退=环境展开 / 推近=聚焦角色 |
| 主体蹲下 | 俯拍下摇（Tilt Down） | 跟随视线下降 |
| 主体跳跃 | 仰拍上摇（Tilt Up）或固定+微仰 | 仰拍=力量感 / 固定=客观视角 |
| 主体静止微动 | 缓慢推近（Slow Dolly In）或固定 | 推近=情感聚焦 / 固定=客观观察 |
| 主体快速奔跑 | 侧面跟拍（Tracking Shot） | 速度感最强 |

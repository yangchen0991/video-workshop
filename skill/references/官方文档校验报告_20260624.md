## 一、官方文档核心规范（一字不漏提取）

### 1. 特殊字符规范表（官方原文）

| 信息类型 | 符号 | 示例 |
|----------|------|------|
| 音乐 | **（）**（全角圆括号） | （背景中播放着快节奏的摇滚乐） |
| 音效 | **<>**（半角尖括号） | <远处传来狗叫声> |
| 台词 | **{}**（半角花括号） | {你好，世界} |
| 字幕 | **【】**（全角方头括号） | 【第一章：启程】 |

### 2. 素材引用规则（官方实战案例验证）

| 任务类型 | 引用方式 | 示例 |
|----------|---------|------|
| 多模态参考 | `@图片1`、`@视频1`、`@音频1`（@前缀） | `@图片1中的女孩作为主角` |
| 视频编辑/延长 | 直接写 `视频1`、`图片1`（不带@、不带尖括号、不加"参考"） | `在视频1的台面上添加炸鸡` |
| 轨道补齐 | 直接写 `视频1...接视频2` | `视频1，树叶落地的瞬间，激起金色粒子特效，一阵风吹过，接视频2。` |
| 主体定义 | `将图片N中[特征]定义为[主体名]` | `将图片1中穿红色连衣裙、戴草帽的女人定义为主体1` |
| 音频参考 | `参考@音频N中的音色` | `参考@音频1中的音色，生成...` |

**关键发现：尖括号 `<>` 在实际提示词中仅用于音效，不用于包裹素材编号。** 官方公式描述中的 `<图片N>` 是文档占位符表示法（"此处替换为实际值"），不是实际提示词要写的符号。

### 3. 进阶公式（官方原文）

```
提示词进阶公式：精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质 + 约束条件
```

### 4. 分镜时序规则（官方原文）

- 使用 `镜头1、镜头2、镜头3` 等标识，按事件发生顺序（先主后次）组织内容
- **不强制限制每段时长，优先让模型根据剧情自然生成节奏**
- **模型对精确时间（如 0–3 秒）的支持不稳定，强行限制时长可能导致生成结果异常**
- 每个镜头按四要素组织：① 运镜或镜头切换方式 → ② 主体动作与表情 → ③ 位置或空间变化 → ④ 音频信息

### 5. 三大抑制句式（官方原文）

- 字幕抑制：`保持无字幕。` / `避免生成任何文字或字幕。`
- Logo 抑制：`不要生成Logo。`
- 水印抑制：`不要生成水印。`

### 6. 情绪具象外化表（官方5项原文）

| 抽象情绪 | 外化表述 |
|----------|---------|
| 悲伤 | 低头、肩膀微微颤抖、眼眶泛红、手指无意识地攥紧衣角、泪水在眼眶里打转但没有落下 |
| 喜悦 | 嘴角抑制不住地上扬、眉眼舒展、脚步变得轻快、下意识地哼起小曲、忍不住原地转个圈 |
| 紧张/焦虑 | 频繁地看手表、手指不停敲击桌面、呼吸急促、眼神闪躲、无意识地啃咬指甲 |
| 愤怒 | 双拳紧握、下颌线紧绷、胸口剧烈起伏、眼神如刀般锐利、从牙缝里挤出话语 |
| 释然 | 长长地舒了一口气、紧绷的肩膀完全放松下来、脸上露出久违的、淡淡的微笑、抬头望向远方 |

### 7. 动作描述三要求（官方原文）

1. **肢体细化 + 程度量化**：动作具体到手/腿/头/肩背，补充幅度/速度/力度
2. **优先低缓连续小动作**：避免狂奔/大跳/剧烈翻滚等高爆发大动态
3. **补充动作过渡衔接**：写明前后动作的惯性与承接关系

### 8. 运镜规则（官方原文）

- 直接使用标准运镜术语（中景、特写、全景、缓慢推镜、平稳横移、固定镜头）
- **一个镜头里尽量只指定1种运镜方式**，不要同时要求推拉摇移

### 9. 轨道补齐限制（官方原文）

- 最多支持 **3 段视频输入**
- **总时长不得超过 15 秒**
- 模型将自动截取首尾视频的衔接部分，仅保留必要片段参与合成

### 10. 防分身约束（官方原文）

```
视频全程禁止出现外形、着装、配饰完全一致的人物，禁止生成同款分身、双胞胎效果，
同一画面中仅保留单个对应人物，不出现人物重复复刻。
```

### 11. 人物参考规范（官方原文）

- 准备**人脸特写大头照**（仅含面部，无表情最佳，尽量减少肩颈、背景干扰）
- **严禁使用人物多视图/三视图**——多视图素材包含同一人物的不同角度，模型易将其识别为多个不同主体
- 参考人物超过 **4 人** 时模型稳定性下降，需分步生成
- 重要素材越靠前权重越高

### 12. 中文发音优化（官方原文）

将易读错文字替换为发音一致的常用同音字（如"螭龙山"→"吃龙山"）。

### 13. 片尾噪音处理（官方原文）

视频片尾出现咔哒声/截断杂音时，重新生成或使用剪映音量包络线做音频淡出。

### 14. 音色参考优化（官方原文）

在提示词中补充细致的音色特征描述（如"低厚温润带细碎颗粒感中年男声"），保持台词风格与参考音频语气相近。

## 四、官方文档额外注意事项

1. **视频延长衔接跳变**：当前无根本解决方案，建议前一段末尾删减6帧+后一段开头删减1帧；续写时以转场切镜时刻结尾
2. **延长视频画质劣化**：建议先转为白模视频再续写，控制续写次数
3. **风格漂移**：需加明确风格约束词，必要时先将参考图变为目标风格
4. **特效不符预期**：建议用参考视频定义特效而非纯文字描述
5. **素材配置推荐**：角色锚定1-2张（大头照+全身照）+ 场景定调1张 + 运镜参考1段 + 节奏氛围1段 = 总计4-5个
6. **横屏字幕概率更低**：业务允许时优先横屏生成，后裁剪为竖屏
7. **视频延长/编辑时不加"参考"**：直接用视频N指代，加"参考"会被误判为参考任务
8. **Asset ID 不能替代素材编号**：模型无法直接关联 Asset ID 与参考内容
9. **空间关系优先用参考图表达**：减少复杂文字描述
10. **台词语言统一**：避免中英文混用（专有名词除外）

## 原则一：动作必须有叙事动机

> 源自：斯皮尔伯格 — "道具即潜台词"、"空间站位即立场"

### 电影原理论
斯皮尔伯格在《少数派报告》中用木球的特写、角色的站位对立、坐下与蹲下的构图变化，让每一个物理动作都承载叙事信息。

### AI 转化规则

| 规则 | 说明 | 违反后果 |
|------|------|---------|
| **动作=叙事** | 每个动作必须回答"这个动作为什么存在？是为了展现权力？亲密？威胁？孤立？"——不能"为了动而动" | 无动机动作=浪费时长+观众困惑 |
| **道具即潜台词** | 关键道具在动作序列中出现≥2次，每次伴随一个微动（如"紧握→放下→再拿起"） | 道具出现一次就消失=镜头浪费 |
| **站位=关系** | 两个角色物理间距<0.5m=亲密/对抗；1-3m=日常对话；>3m=疏离/对立 | 无意义站位=角色关系模糊 |

### Seedance 实现

```
❌ 差：角色A和角色B在房间里说话。
✅ 好：角色A站在窗边（背光=权力不确定），角色B坐在房间中央的椅子上（被光包围=自信）。两人间距3m（视觉对立）。
   角色A从窗边走到B面前（间距从3m→0.5m=从对立到亲密），弯腰手扶B的椅背（身体语言=掌控局势）。
```

## 原则二：极简运动放大每一寸变化

> 源自：库布里克 — "单帧如照片"、"极简运动"

### 电影原理论
库布里克在《发条橙》中让三个 Droogs 呈三角形静止站位，Georgie 仅伸出一条腿、拐杖尖端指向，所有叙事在静态构图中完成。

### AI 转化规则

| 规则 | 说明 | 违反后果 |
|------|------|---------|
| **静态构图优先** | 60-70% 的镜头应为静态或仅含微动（呼吸、视线转移、阴影变化），大动作留给情绪爆发点 | 全程都在动=视觉疲劳+观众不知道看哪里 |
| **微动即叙事** | 一个眼神转移、一次手指微曲、一次喉结滚动——比一次大挥手更有叙事分量 | 忽视微动=丧失了 AI 最擅长的细节表现力 |
| **大动作限量** | 15s 内剧烈动作链≤3个，每个大动作前必须有≥2秒的静态/微动积蓄 | 动作过载=画面撕裂+模型崩溃 |

### Seedance 实现

```
❌ 差：角色A站起来→走到窗前→转身→坐下→拿起杯子→喝水→放下杯子（15s内7个动作）
✅ 好：镜头1（4s）：角色A静坐桌前，指尖缓缓划过桌面纹理（微动），视线落在窗外的雨上。
        镜头2（8s）：角色A缓慢起身（KP0→KP4，4s），走到窗前（A02行走，4s），
                   手扶窗框（微动），额头轻靠玻璃（微动）。
        镜头3（3s）：角色A的侧脸特写，窗玻璃上的雾气随呼吸晕开（微动），一滴雨沿窗滑落。
```

## 原则三：角色动作驱动运镜方向

> 源自：伊纳里图 — "演员动作=镜头动机"

### 电影原理论
伊纳里图在《鸟人》一镜到底中，Sam 的移动直接引导了镜头的跟摇和推移，角色不看哪里镜头就不去哪里。

### AI 转化规则

| 规则 | 说明 | 违反后果 |
|------|------|---------|
| **动作=运镜动机** | 主体向右走→镜头跟摇向右；主体蹲下→镜头俯拍下摇；主体回头看→镜头反打 | 动作和运镜方向不一致=观众方向迷失 |
| **一个镜头一个运镜** | 每个镜头只给一个主导运镜（Seedance 2.0 规范），多种运镜同时要求最容易失稳 | 多运镜=画面抖动/撕裂 |
| **视线即引导** | 角色看向某处→观众视线随之转移→下一个镜头切到那个方向 | 视线不引导=观众不知道下一个镜头为什么在那里 |

### Seedance 实现

```
❌ 差：镜头慢慢推近同时角色转身，然后镜头摇到窗外。
✅ 好：【镜头1】：角色A转身面向右侧（KP0→KP4），镜头跟摇向右（Pan Right），
       停在角色A右侧面特写，A视线望向画面右侧外（引导观众期待）。
      【镜头2】：切到A视线方向的窗外景色（视线引导决定切镜逻辑）。
```

## 原则四：动作拆解成关键姿态链

> 源自：搜狐 AI 视频四步法 — "大动作→3-5个关键姿态"

### 电影原理论
传统 2D 动画的关键帧（Keyframe）思维：只定义极限姿态，中间帧由补间自动生成。AI 视频同理——只定义 KP，不写中间过渡。

### AI 转化规则

| 规则 | 说明 | 违反后果 |
|------|------|---------|
| **3-5个KP** | 每个动作拆解为 3-5 个关键姿态，相邻 KP 间姿态变化量≤30cm（身体任意部位） | KP过多=指令混乱/KP过少=跳帧 |
| **预备动作不可省** | 每个大动作前 0.5-1s 的预备微动（如转头前的视线先移、跳跃前的下蹲）必须明确写出 | 无预备=无重量感，AI模型缺乏力学约束 |
| **回收动作不可省** | 大动作后 0.3-0.5s 的回收/缓冲/姿态调整不可省略 | 无回收=动作停在"半空中"，不自然 |

### Seedance 实现

```
❌ 差：角色A站起来。
✅ 好：KP0：坐姿，双手放膝上
      KP1：双手撑椅子扶手，身体前倾15°，重心前移（预备）
      KP2：臀部离开椅面5cm，膝盖伸展中，双手持续发力
      KP3：完全站直，膝盖锁紧，双手离开扶手自然下垂
      KP4：微调整站姿，双肩后收，挺胸（回收/收束）
```

## 原则五：节奏-情绪精准对位

> 源自：搜狐 AI 视频四步法 — "节奏=动作快慢×镜头停留时长"

### 电影原理论
动作的节奏传递情绪。快的=紧张/焦虑/兴奋；慢的=沉思/悲伤/庄严；快慢反差=震惊/情绪转折。

### AI 转化规则

| 节奏类型 | KP间距 | 镜头停留 | 动作量 | 适用情绪 |
|---------|--------|---------|--------|---------|
| **快切型** | 0.3-0.5s | 0.5-1s | 多动作 | 战斗/追逐/恐慌/兴奋 |
| **叙事型** | 0.8-1.2s | 1.5-3s | 中等 | 日常对话/情感流露/行动 |
| **长镜型** | 1.5-2.5s | 3-5s | 少动作 | 史诗/沉思/孤独/庄严 |
| **反差型** | 由快转慢 | 2s+停留 | 快→突然静止 | 震惊/情绪爆发后余韵 |

### Seedance 实现

```
快切型示例：
  镜头1（1s）：拳面冲击目标，面部肌肉抖动 ← KP间距0.3s
  镜头2（0.8s）：对手身体后仰，唾液飞溅 ← KP间距0.3s

反差型示例：
  镜头1（3s）：快切打斗（如上）
  镜头2（5s）：胜利者缓缓跪下，手撑地面，大口喘气，背景慢动作飘落的碎屑
             ← KP间距2s，镜头停留5s
```

## 原则六：微观物理强化真实感

> 源自：CSDN Seedance 动作捕捉 — "物理反馈是真实感的核心"

### AI 转化规则

| 规则 | 说明 |
|------|------|
| **冲击反馈** | 拳打到目标→目标表面产生波纹/凹陷；脚落地→地面微尘扬起；物体掉落→弹跳+尘土 |
| **衣物物理** | 衣物随动作产生褶皱/飘动/拉伸；风吹→衣物向后飘；旋转→衣物离心外摆 |
| **头发物理** | 转头→发丝惯性滞后；跳跃→头发上浮后下压；奔跑→头发向后飘动 |
| **重量感** | 重物被拿起→手臂肌肉紧绷+身体重心变化；轻物→轻松提起无重心偏移 |
| **呼吸系统** | 动作后呼吸急促→胸腔起伏频率增加；静态时呼吸平稳→胸腔微动 |

## 原则七：动作序列的情绪弧线

> 源自：三源融合 — 搜狐节奏理论 + StudioBinder 电影理论 + CSDN 物理参数

### 全片动作情绪弧线模板

```
积蓄期（0-40%时长）
    ↓ 微动为主，静态构图，大量视线和呼吸细节
    ↓ 每15s内 ≤1个大动作
    ↓ 氛围：日常/悬疑/压抑/期待

转折期（40-60%时长）
    ↓ 动作量上升
    ↓ 节奏从中速过渡到快速
    ↓ 预备动作→爆发动作

释放期（60-85%时长）
    ↓ 动作高峰：快速连续动作链
    ↓ 每15s内 1-2个大动作
    ↓ 氛围：战斗/追逐/情绪爆发

余韵（85-100%时长）
    ↓ 动作骤减，回到静态或微动
    ↓ 长镜头停留3-5s+
    ↓ 氛围：反思/释放/悲痛/宁静
```

## 快速参考：动作编排自检清单

每个动作序列完成后逐项检查：

- [ ] 这个动作为什么存在？（叙事动机）
- [ ] 关键姿态 3-5 个，相邻位移≤30cm？
- [ ] 有预备动作吗？（大动作前≥0.5s微动）
- [ ] 有回收动作吗？（大动作后≥0.3s缓冲）
- [ ] 剧烈动作链 ≤3个/15s？
- [ ] 动作方向与运镜方向互锁？
- [ ] 所有动作含三维量化（幅度/速度/力度）？
- [ ] 情绪已外化为身体细节（非抽象词）？
- [ ] 冲击反馈已描述（尘土/波纹/衣物反应）？
- [ ] 60-70% 时长为静态/微动？
- [ ] 呼吸节奏与动作节奏匹配？
- [ ] 节奏类型与目标情绪一致？
